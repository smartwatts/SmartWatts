# SmartWatts User Guide

## Overview

Welcome to SmartWatts! This guide will help you get started with monitoring and managing your energy consumption.

## Getting Started

### 1. Account Creation

#### Register Account
1. Visit https://mysmartwatts.com/register
2. Enter your information:
   - Username
   - Email address
   - Phone number
   - Password
3. Click "Register"

#### Verify Email
1. Check your email for verification link
2. Click the verification link
3. Your email is now verified

#### Verify Phone
1. Click "Verify Phone" in your profile
2. Enter the 6-digit code sent to your phone
3. Your phone is now verified

### 2. Dashboard Overview

#### Main Dashboard
- **Energy Consumption**: Real-time energy usage
- **Cost Tracking**: Current and projected costs
- **Device Status**: Status of all connected devices
- **Alerts**: Important notifications

#### Navigation
- **Dashboard**: Main overview
- **Devices**: Device management
- **Analytics**: Energy analytics
- **Billing**: Cost and billing information
- **Settings**: Account settings

### 3. Adding Devices

#### Add Device
1. Go to "Devices" page
2. Click "Add Device"
3. Select device type:
   - Smart Meter
   - Solar Inverter
   - Generator Monitor
   - Smart Plug
   - CT Clamp
4. Enter device information
5. Click "Add Device"

#### Device Activation
1. Connect device to power
2. Scan QR code or enter activation code
3. Wait for device activation
4. Device is now connected

## Troubleshooting

### Common Issues

#### Device Not Connecting
1. Check device power
2. Verify network connection
3. Check activation code
4. Contact support

#### Data Not Updating
1. Check device status
2. Verify network connection
3. Refresh dashboard
4. Contact support

#### Account Issues
1. Verify email address
2. Check password
3. Reset password if needed
4. Contact support
