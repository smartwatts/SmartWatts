# Troubleshooting Guide

## Common Issues

### Cloud Run Deployment Failures

**Issue**: Service fails to start
**Solution**: Check logs, verify environment variables, check Cloud SQL connection

### Database Connection Issues

**Issue**: Cannot connect to Cloud SQL
**Solution**: Verify Cloud SQL Proxy, check IAM permissions, verify connection string

### Secret Manager Access

**Issue**: Cannot access secrets
**Solution**: Verify service account has Secret Manager accessor role
